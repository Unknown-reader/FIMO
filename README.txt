Консольная утилита для ОС Linux реализующая функционал контроля целостности 
содержимого каталога файлов FIMO (File Integrity MOnitoring).

Для постановки каталога на учет использовать:
./fimo [-s set] 'path/to/dir' 'path/to/control-file

Для проверки целостности каталога использовать
./fimo [-c check] 'path/to/dir' 'path/to/control-file

Особенности реализации/краевые моменты/точки роста:

- файл контроля целостности не должен находиться в отслеживаемой директории
на момент постановки ее на учет и на момент вызова проверки целостности. Причина:
не доработан факт того, что если файл целостности находится в отслеживаемой директории
на момент его заполнения, то хэш-сумма для него будет посчитана, как для
пустого файла.

- переименованные файлы, над которыми не были произведены действия (хэш-сумма
не изменилась) утилита считает не нарушающими целостность, однако информация
о переименовании логируется.

- утилита корретктно обрабатывает подкаталоги, находящиеся в отслеживаемом, рекурсивно
обрабатывая их.

- алгоритм сравнения содержимого файла целостности и актуального содержимого каталога
не является оптимальным: временная сложность O(2*n*m + n + m), где n и m - количество
файлов в директории и файле целостности соответственно. Вероятно, значительно более лучшим
решением была бы попытка использования хэш-таблицы.
